# PyCAM: Toolpath Generation for 3-Axis CNC machining


## Usage

Extract the archive or checkout the sources.

Graphical Interface: `scripts/pycam`

Scripted Toolpath Processing: `scripts/pycam-flow FLOW_SPECIFICATION_FILE`
